# P2P Chat Enterprise

A premium peer-to-peer chat application built with SvelteKit.

## Features

- **P2P Communication** - Direct peer-to-peer messaging
- **QR Code Auth** - Scan to connect
- **Enterprise Premium** - Subscription features
- **Modern UI** - Dark theme with gradients

## Routes

- `/` - Home/Auth page
- `/chat` - P2P Chat interface
- `/premium` - Premium features

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- Svelte 5 (Runes)
- SvelteKit 2
- Node.js
- Tailwind-inspired styling
